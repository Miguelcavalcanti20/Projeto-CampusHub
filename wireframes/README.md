# Wireframes

Os wireframes foram utilizados durante a fase inicial de planejamento do projeto CampusHub para definir a estrutura das telas e a navegação do aplicativo.

Durante o desenvolvimento, o layout sofreu adaptações e melhorias em relação ao planejamento inicial, resultando na versão final apresentada no aplicativo.

As telas implementadas incluem:

* Login
* Home
* Calendário
* Avisos
* Perfil
* Configurações
