# CampusHub

## Sobre o Projeto

O CampusHub é um aplicativo móvel desenvolvido para unir informações acadêmicas, eventos institucionais, palestras, workshops e avisos em uma única plataforma.

## Objetivo

Facilitar o acesso dos estudantes às informações divulgadas pela instituição de ensino.

## Tecnologias Utilizadas

- React Native
- JavaScript
- Expo
- AsyncStorage
- Git e GitHub

## Funcionalidades

- Login
- Listagem de eventos
- Confirmação de presença
- Calendário acadêmico
- Avisos institucionais
- Perfil do usuário
- Tema claro e escuro
