# Descrição do Projeto

## CampusHub

O CampusHub é um aplicativo móvel desenvolvido para unir informações acadêmicas e eventos institucionais em uma única plataforma.

A proposta surgiu da necessidade de facilitar o acesso dos estudantes a informações relacionadas: palestras, workshops, feiras de estágio e eventos acadêmicos. Em muitas instituições de ensino, essas informações são divulgadas por diferentes canais, dificultando o acompanhamento pelos alunos.

O aplicativo oferece uma interface simples e intuitiva que permite visualizar eventos futuros, consultar avisos, acompanhar o calendário acadêmico, acessar informações do perfil do usuário e personalizar a experiência por meio dos temas claro e escuro.

### Tecnologias Utilizadas

* React Native
* JavaScript
* Expo
* AsyncStorage
* React Native Components

### Funcionalidades Principais

* Visualização de eventos
* Confirmação de presença em eventos
* Consulta de avisos institucionais
* Calendário acadêmico
* Perfil do usuário
* Tema claro e escuro com persistência local
