# Requisitos Não Funcionais

RNF 1 – O aplicativo deve ser desenvolvido utilizando React Native.

RNF 2 – O aplicativo deve ser compatível com dispositivos Android.

RNF 3 – A interface deve ser intuitiva e de fácil utilização.

RNF 4 – O sistema deve apresentar tempo de resposta adequado para navegação.

RNF 5 – As preferências do usuário devem ser armazenadas localmente utilizando AsyncStorage.

RNF 6 – O aplicativo deve possuir suporte aos temas claro e escuro.

RNF 7 – O código-fonte deve ser organizado e documentado.

RNF 8 – O projeto deve ser mantido em repositório GitHub para controle de versão.
